#### 一、下载asciinema

```shell
sudo apt install asciinema
```

注意：

因为第一次用官网的命令下载出错，所以换了一台虚拟机，并下载的是Ubuntu自带的asciinema

---

#### 二、asciinema关联本地和在线账号

```shell

```
